# IMDb w/ class component
Simple responsive react app created using react class components. 
#### Features:
* Pagination
* Sorting
* Filtering

### Run at local environment
* With `npm`
```bash
npm install
npm start
```
* or with `yarn`
```bash
yarn
yarn start
```
